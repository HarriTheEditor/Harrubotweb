Commands

hb info help - Tells you the commands.

hb help - Exactly the same as info help, just shortened.

hb info prefix - Tells you the prefix for the bot.

hb info version - Current version of the bot.

hb hello - A little friendly hello!

hb meme - Get a meme!

hb furret - Get a image of a furret!

hb joke - Dad jokes.

hb gay - no you

hb kiss - Enter a username to kiss someone.

hb hug - Enter a username to hug someone.

hb kill - Do I really need to tell you?

hb apologize - Apologize to someone using a bot... original I guess.

There is more commands, but these are the main ones.
